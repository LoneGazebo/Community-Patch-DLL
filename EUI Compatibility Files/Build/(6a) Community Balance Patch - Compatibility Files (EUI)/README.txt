== Description ==

This file updates all relevant LUA changes from the CBP (and all accompanying mods) to be compatible with EUI.

== Installation Instructions ==

Make sure you delete the LUA folders in the CP and CBP mods before running the game!

== Potential Mod Conflicts ==

None, but you must be using EUI, CBP, CSD, C4DF and More Luxuries for this file to be used.
